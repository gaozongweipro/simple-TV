# SimpleTV 使用说明

## 适用设备

本应用面向低性能横屏安卓设备，第一版按安卓 8 及 800x400 左右触屏设备设计。当前操作方式只考虑触屏，不需要遥控器。

## 基本操作

- 点按播放画面：打开或关闭右侧频道列表。
- 在右侧频道列表上下滑动：滚动频道。
- 点按频道名称：切换频道。
- 长按播放画面：打开设置。
- 设置里的“刷新频道”：从远程频道地址重新拉取频道清单。
- 设置里的“恢复默认”：清除本地缓存，回到 APK 内置频道。

## 默认频道

默认频道文件位于：

```text
app/src/main/assets/default_channels.json
```

当前只内置央视和卫视。APK 覆盖安装后，如果内置频道版本号高于本地缓存版本，会优先使用新的内置频道，避免继续播放旧版本缓存源。

## 远程频道更新

应用支持远程更新频道清单。远程 JSON 的结构与内置文件一致：

```json
{
  "version": 20260516,
  "updatedAt": "2026-05-16",
  "channels": [
    {
      "id": "cctv13",
      "name": "CCTV-13 新闻",
      "group": "央视",
      "sources": [
        {
          "url": "http://example.com/live/cctv13/index.m3u8",
          "label": "默认"
        }
      ]
    }
  ]
}
```

字段说明：

- `version`：频道清单版本号。数字越大越新。
- `updatedAt`：更新时间，仅用于人工识别。
- `channels`：频道列表。
- `id`：频道唯一标识，不要重复。
- `name`：界面显示名称。
- `group`：频道分组，可写 `央视` 或 `卫视`。
- `sources`：播放源列表。播放器会按顺序尝试，失败后自动切到下一个源。
- `url`：HLS 直播地址，建议使用 `.m3u8`。
- `label`：源名称，例如 `默认`、`备用`。

## 播放显示

播放器会按视频原始比例居中显示，不会强行拉伸变形。屏幕比例与视频比例不一致时，会自动留黑边。

## 常见问题

### 有声音没图像

通常是频道源本身只有音频流，或者缓存里还保留旧源。先长按画面进入设置，点“恢复默认”。如果仍然存在，需要替换该频道源。

### 有图像没声音

可能是直播源缺少音频轨，或设备音量、音频焦点异常。当前播放器已按媒体播放方式申请音频焦点，并把音量设为 1。

### 播放卡顿

直播源可能跨地区或码率过高。优先使用普通清晰度 HLS 源，不建议默认使用 4K 源。当前播放器增加了缓冲时间，以降低弱网下的卡顿。

### 频道列表不能滚动

频道列表已经使用可滚动容器。打开右侧列表后，在列表区域上下滑动即可滚动。
